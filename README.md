# Bank Management System (Python + SQLite)

## Description
This project implements a **Bank Management System** in Python using SQLite.  
It demonstrates full **CRUD operations** (Create, Read, Update, Delete) on bank accounts, along with account searches and simple transaction management.  

Developed as a **DBMS project** during undergraduate studies.

## Features
- Add new bank accounts with customer details and initial deposit
- View individual or all account details
- Update customer information and perform deposit/withdraw
- Delete/close an account
- Search accounts by account number or customer name
